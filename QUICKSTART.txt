📦 核心文件（必须上传）
├── index.html          (10KB)  - 主页面
├── style.css           (18KB)  - 样式文件
├── app.js              (20KB)  - 应用逻辑
└── qrcodegen.js        (22KB)  - 二维码库

总大小：~70KB（压缩后 ~25KB）

================================================================================
🚀 部署方式（选择其一）
================================================================================

方式1：GitHub Pages（最简单，免费）
--------------------------------------
1. 创建 GitHub 仓库
2. 上传 4 个核心文件
3. Settings → Pages → 启用
4. 访问 https://你的用户名.github.io/仓库名

方式2：Vercel（推荐，免费HTTPS）
--------------------------------------
1. 访问 vercel.com
2. 导入 GitHub 仓库或上传文件
3. 自动部署完成
4. 获得免费域名和HTTPS

方式3：自己的服务器
--------------------------------------
# 上传文件
scp index.html style.css app.js qrcodegen.js user@server:/var/www/html/

# Apache 服务器（同时上传 .htaccess）
sudo systemctl reload apache2

# Nginx 服务器（参考 nginx.conf 配置）
sudo systemctl reload nginx

🎯 生产环境优化（可选）
================================================================================

   - Cloudflare CDN

   - CSS/JS: 缓存 7 天

3. 启用压缩
   - Gzip 自动启用（配置文件已设置）
   - 可减少 70% 传输大小

================================================================================
🆘 常见问题
================================================================================

Q: 页面打不开？
A: 检查文件路径和权限（chmod 755）

Q: 功能不正常？
A: 打开浏览器控制台（F12）查看错误

Q: 手机端显示异常？
A: 确保 viewport meta 标签存在

Q: 下载不工作？
A: 检查浏览器是否阻止弹窗

Q: 需要数据库吗？
A: 不需要！完全静态，无需后端
